# Инструкция по работе с настройками сайта

## Значения по умолчанию

В БД добавлены значения по умолчанию для настроек сайта:

- **Описание деятельности**: Мы предлагаем высококачественные мебельные решения для вашего дома и офиса. Надежные материалы, современный дизайн и индивидуальный подход к каждому заказу.
- **Телефон**: +7 (999) 123-45-67
- **Email**: info@mebel-site.ru
- **Адрес**: г. Москва, ул. Примерная, д. 1
- **Ссылка на Яндекс.Карты**: https://yandex.ru/maps

## Способы сброса настроек к значениям по умолчанию

### Способ 1: Использование фикстуры

Загрузить значения по умолчанию из фикстуры:

```bash
python manage.py loaddata default_settings.json
```

Фикстура находится в: `my_site/fixtures/default_settings.json`

### Способ 2: Использование миграции

Миграция `0006_load_default_settings.py` автоматически применяется при выполнении команды:

```bash
python manage.py migrate
```

### Способ 3: Программный сброс

Создайте скрипт для сброса настроек:

```python
from my_site.models import SiteSettings

settings = SiteSettings.objects.first()
if settings:
    settings.business_description = "Мы предлагаем высококачественные мебельные решения для вашего дома и офиса. Надежные материалы, современный дизайн и индивидуальный подход к каждому заказу."
    settings.phone = "+7 (999) 123-45-67"
    settings.email = "info@mebel-site.ru"
    settings.address = "г. Москва, ул. Примерная, д. 1"
    settings.yandex_map_link = "https://yandex.ru/maps"
    settings.save()
```

## Дополнительная информация

- Настройки сайта реализованы как Singleton (одна запись в БД)
- Модель `SiteSettings` в файле `my_site/models.py`
- Админка настроек в файле `my_site/admin.py`
- Фикстура для загрузки значений по умолчанию в `my_site/fixtures/default_settings.json`
